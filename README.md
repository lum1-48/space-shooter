# Space Shooter

Браузерная аркадная игра на чистом HTML/CSS/JavaScript. Никаких зависимостей — один файл `index.html`.

## Запуск

Открой `index.html` в браузере. Поддерживается установка как PWA (без интернета, иконка на рабочем столе).

## Управление

| Клавиша | Действие |
|---------|----------|
| `W A S D` | Движение |
| `Escape` / `⏸` | Пауза |

Корабль стреляет автоматически.

## Враги

| Модель | Цвет | Особенность | HP |
|--------|------|-------------|----|
| Истребитель | Красный | Передне-стреловидные крылья, быстрый | 1 |
| Перехватчик | Оранжевый | Дельта-крыло, волнообразное движение | 1 |
| Жук-эмиттер | Зелёный | Выпускает крест-лучи по диагонали, 2 HP | 2 |
| Канонир | Синий | Бронированный, стреляет в игрока | 3 |

Появляются постепенно: с уровня 1 — красный и оранжевый, с уровня 2 — зелёный, с уровня 3 — синий.

## Боссы

| Модель | Порог | Атака | HP |
|--------|-------|-------|----|
| Краб (мини-босс) | 500 | Веер из 5 пуль | 25 |
| Пришелец (Boss I) | 1000 | Широкий веер из 7 пуль | 60 |
| Демон (Boss II) | 2500 | Спираль + два туррета | 140 |
| Дредноут (Boss III) | 5000 | Прицельный веер + кольцо | 260 |

Boss II вызывает два синих туррета по бокам экрана.

## Усиления

После каждого убитого босса выбери одно:

- **Тройной выстрел** — три пули вместо одной
- **Дополнительная жизнь** — +1 сердце (максимум 5)
- **Напарник** — зелёный корабль рядом, 2 HP, быстрая атака; накапливаются пирамидой

## Рекорды

Топ-5 результатов (очки, уровень, дата) сохраняются в `localStorage`.

## PWA

Сайт поддерживает установку как Progressive Web App:
- **Android**: Chrome → меню → «Добавить на главный экран»
- **iOS**: Safari → «Поделиться» → «На экран "Домой"»

Игра работает офлайн после первого открытия.
