# Space Shooter

Браузерная аркадная игра на чистом HTML.

## Управление

| Клавиша | Действие |
|---------|----------|
| `W A S D` | Движение |
| `Escape` / `⏸` | Пауза |

## Геймплей

- Корабль стреляет автоматически
- Убивай врагов, набирай очки
- После каждого босса выбирай усиление

## Враги

| Цвет | Особенность |
|------|-------------|
| Красный | Быстрый, 1 HP |
| Оранжевый | Волнообразное движение, 1 HP |
| Синий | Стреляет в игрока, 3 HP |

## Боссы

| Босс | Очки | Атака |
|------|------|-------|
| Мини-босс | 500 | Веер из 5 пуль |
| Boss I | 1000 | Широкий веер |
| Boss II | 2500 | Спираль |
| Boss III | 5000 | Прицельный + кольцо |

## Усиления

После каждого босса выбери одно:

- **Тройной выстрел** — три пули вместо одной
- **Дополнительная жизнь** — +1 сердце (максимум 5)
- **Напарник** — зелёный корабль рядом; 2 HP, быстрая атака. Накапливаются пирамидой

## Рекорды

Топ-5 результатов сохраняются в браузере (localStorage).

## Запуск

Открой `index.html` в браузере — никаких зависимостей и сборки не нужно.
